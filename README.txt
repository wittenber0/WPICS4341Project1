Instructions:
To run this program, extract the program zip files. Copies of the source code and graphs are also included in this directory, however you will need only the fiel "SearchingAI.jar" and a formatted graph file of your choosing. You will need the file path names for both the executable jar and the graph file.

Open a command prompt and type in the command:
    java -jar <Executable File Path Name> <Graph File Path>
